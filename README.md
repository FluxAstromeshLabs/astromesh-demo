# astromesh-demo

Install and start "fluxd" chain, the multiVM chain with EVM, WASM and SVM integrated

## Setup environment

1. Docker

```
./start-docker.sh
```

2. Don't have docker? Install dependencies and run binary directly:

```
./start.sh
```

## Run examples

On another terminal: 
```
yes 12345678 | go run examples/<something>
```
